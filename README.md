# Rock Paper Scissors ✊✋✌️  

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)  
![Status](https://img.shields.io/badge/Status-Active-brightgreen.svg)  
![License](https://img.shields.io/badge/License-MIT-orange.svg)  

A simple and fun command-line **Rock–Paper–Scissors** game built in **Python**.  
This project demonstrates the use of randomization, condition handling, and user interaction in a clean and beginner-friendly way.  

---

## 🔹 Features  
| Feature | Description |
|---------|-------------|
| 🤖 Human vs Computer | Play against an AI opponent |
| 🎲 Randomized Moves | Computer chooses randomly each round |
| ✅ Input Validation | Ensures only valid choices are accepted |
| 🏆 Results | Win / Lose / Draw clearly shown |

---

## 🔹 Installation & Setup  
> ⚡ **Requirement:** Python 3.x must be installed  

```bash
# Clone this repository
git clone https://github.com/Tummepallisivanagalakshman/rock-paper-game.git
cd rock-paper-game

# Run the game
python3 game.py
Your move (rock/paper/scissors): rock
Computer chose: scissors
✅ You win! Rock beats scissors.
Play again? (y/n): y
